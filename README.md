# Executive Product Leadership Portfolio

## Purpose
This repository is the entry point to a structured portfolio of executive product leadership systems.

It brings together a set of related repositories that demonstrate:
- systems thinking
- decision quality
- AI fluency
- market awareness
- applied product execution

---

## What This Repo Is

- A portfolio index and navigation layer
- A guide to how the repositories connect
- A narrative frame for recruiters, hiring managers, and peers
- A structured overview of how I think, operate, and build as a product leader

---

## What This Repo Is NOT

- A standalone knowledge repository
- A code repository
- A duplicate of the linked repositories
- A generic personal GitHub homepage

---

## Who This Is For

- Hiring managers evaluating executive product leadership capability
- Recruiters seeking a clear portfolio entry point
- Product leaders interested in systems, AI, and modern product execution
- Collaborators looking for a structured view of my work

---

## Portfolio Logic

This portfolio is organized as a connected system:

**Signal → Thinking → Tools → Build**

- **Signal** → external market inputs and curated intelligence  
- **Thinking** → mental models and decision frameworks  
- **Tools** → AI models, tools, and capability fluency  
- **Build** → experiments, prototypes, and AI-assisted execution  

This structure is designed to reflect how modern executive product leaders learn, decide, and operate.

## How the Portfolio Works

Each repository is designed to stand on its own, but the portfolio is strongest when viewed as a connected ecosystem.

### Signal Center
Provides the external signals, ideas, and sources that inform what matters.

### Mental Models
Provides the frameworks and structured thinking used to interpret those signals.

### AI Fluency
Provides the capability layer for understanding the tools and technical landscape.

### Experimentation Lab
Provides the proof layer that shows ideas translated into tangible outputs.

### PLOS
Provides the system-level architecture that anchors the overall philosophy.
